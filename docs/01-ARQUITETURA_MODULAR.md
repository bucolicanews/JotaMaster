# Arquitetura - Jota Master (Core)

## Descrição
O Jota Master é a "Placa-Mãe" do sistema. Ele é responsável pela autenticação (Supabase), roteamento dinâmico, orquestração de agentes de IA (Gemini) e gestão de módulos (Legos).

## Módulos Planejados (Plugins)
1. **Precificação e Viabilidade (Ativo)**
2. **Controle de Clientes (CRM - Planejado)**
3. **Auditoria Fiscal (Em transição para módulo)**

## Dependências
- React, Tailwind, Supabase JS Client, Gemini SDK, n8n Webhooks.
